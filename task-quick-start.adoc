= 入門工作流程
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
透過設定 Google Cloud，然後建立系統來開始使用Google Cloud NetApp Volumes 。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"] link:task-set-up-gcnv.html["設定Google Cloud NetApp Volumes"]
[role="quick-margin-para"]
從 Google 建立一個服務帳戶。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"] link:task-create-working-env.html["新增系統"]
[role="quick-margin-para"]
在NetApp Console中，按一下 *新增系統*，然後按一下 *Google Cloud Platform*，然後按一下 * Google Cloud NetApp Volumes*，然後按一下 *發現*。提供有關服務帳戶和 Google Cloud 項目的詳細資訊。
